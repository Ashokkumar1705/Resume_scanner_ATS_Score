# PRD - AI Resume Analyzer

## Purpose
Explain the product idea and problem.

## Project Name
AI Resume Analyzer

## Problem
Students and job seekers often don't know if their resumes are optimized for Applicant Tracking Systems (ATS), leading to missed opportunities despite being qualified.

## Target Users
- Students
- Job seekers
- Career changers

## Core Features
- **Resume Upload**: Support for text extraction from resumes.
- **AI Analysis**: Deep analysis of resume content using Gemini AI.
- **ATS Score**: A numerical score representing how well the resume matches common ATS criteria.
- **Improvement Suggestions**: Actionable feedback on how to improve the resume.

## User Flow
Landing Page → Signup/Login → Dashboard → Upload Resume → AI Analysis → Detailed Report
