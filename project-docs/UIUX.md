# UI/UX Design

## Aesthetic: Technical Dashboard
- **Mood**: Professional, precise, information-dense.
- **Palette**: Zinc and Slate for a clean, modern look. Emerald for positive scores, Amber for warnings.
- **Typography**: Inter for UI, JetBrains Mono for data points.

## Layout Patterns
- **Split Layout**: Hero section with a clear CTA on the left, visual preview on the right.
- **Bento Grid**: Dashboard cards showing stats and recent reports.
- **Stepped Process**: Clear 1-2-3 flow for uploading and analyzing.

## Interactions
- Smooth transitions using Framer Motion.
- Loading states with progress bars for AI analysis.
- Hover effects on report cards for quick actions.
