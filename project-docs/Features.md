# Features List

## 1. Authentication
- Secure user signup and login using Firebase Authentication (Google Login).
- Persistent sessions for returning users.

## 2. Dashboard
- Overview of previous resume analyses.
- Quick access to start a new analysis.

## 3. Resume Upload & Processing
- Simple interface to paste resume text or (future) upload files.
- Real-time status indicators during processing.

## 4. AI Analysis Engine
- Integration with Gemini 1.5 Flash for rapid, intelligent resume parsing.
- Evaluation of keywords, formatting, and impact statements.

## 5. Detailed Reports
- ATS Score (0-100).
- Section-by-section breakdown (Contact, Experience, Skills, Education).
- Specific "Fixes" and "Enhancements" suggestions.
